Game Interface Guide

  • Click: Interact with buttons and menus
  • ESC: Open/close System menu
  • L: Open/close Questlog
  • P: Open/close Spellbook
  • Drag & Drop: Move spells to action bar
  • 1-9: Use spells assigned to action bar
  • Alt + 1-9: Use secondary spells
  • Enter: Send chat message
  • +: Add tabs in the chat
